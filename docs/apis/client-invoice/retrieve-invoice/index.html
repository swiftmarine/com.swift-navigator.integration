    Retrieve an Invoice

    Introduction

    Retrieve the full JSON representation of a specific invoice or credit note associated with the authenticated client.

    This endpoint is typically used to:

    • Inspect invoice details at line level
    • Validate amounts, taxes, and dates
    • Reconcile invoices with accounting or ERP systems
    • Retrieve structured data behind an invoice reference number

    Technical Details

    Endpoint

    GET /oapi/ci/invoice/{invoiceReferenceNumber}

    GET /oapi/ci/invoice/{invoiceReferenceNumber} HTTP/1.1

    Summary

    The Retrieve Invoice endpoint provides the complete, structured JSON payload for a single invoice or credit note.

    It is intended for:

    • Detailed invoice inspection
    • Accounting and ERP reconciliation
    • Validation of totals, VAT, and line items
    • Programmatic access to invoice metadata and documents
    • The endpoint is synchronous, read-only, and operates strictly within the scope of the authenticated client.
